## How to Solve SQL Injection Labs

1. Identify input parameters.
2. Test with single quote (').
3. Determine database behavior.
4. Find number of columns.
5. Perform UNION attack.
6. Extract data.
7. Document findings.

## Mitigation

- Prepared Statements
- Parameterized Queries
- Input Validation
- Least Privilege Database Accounts
